# Diana & Julian – Hochzeitswebsite 💍

Website für die Hochzeit am **29. Mai 2026**.

## Struktur

```
hochzeit/
├── index.html          ← Hauptseite (Root für GitHub Pages)
├── styles/
│   └── main.css
├── utils/
│   └── main.js
├── assets/             ← Nur verwendete Bilder
│   ├── icon/
│   ├── moods/
│   └── ...
└── pages/
    └── index.html      ← Original (Entwicklungsversion)
```

## Hosting via GitHub Pages

1. Neues Repository auf [github.com](https://github.com) erstellen
2. Diesen Ordner als Git-Repo initialisieren und pushen
3. In den Repository-Einstellungen → **Pages** → Source: **main branch / root**
4. Die Seite ist erreichbar unter: `https://<dein-username>.github.io/<repo-name>/`
